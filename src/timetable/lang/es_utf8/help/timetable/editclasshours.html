Editando horas de clase

Desde esta página puede añadir y eliminar las horas de clase que esta institución está ofreciendo.

Añadir una hora de clase - desde esta sección se puede añadir una nueva hora de clase.
Debe introducir hora de inicio y hora final y clicar el botón Añadir una nueva hora de clase y esta hora se añadirá en la lista de abajo.
Hora final debe ser mayor que hora de inicio, sinó se mostrará un mensaje de error en la parte superior de las secciones.
Por espacios de tiempo de veinte minutos entre horas se mostrará una línea blanca.

Eliminar hora de clase - En la parte inferior de la sección para añadir una hora de clase encontrará una lista ordenada de las horas de clase añadidas, sinó se mostrará un mensaje que indicará que no se han añadido horas de clase.
Para cada fila hay un botón de borrar.
