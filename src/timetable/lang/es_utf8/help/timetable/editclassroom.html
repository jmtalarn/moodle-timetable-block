Editando las aulas

Desde esta página puede añadir, modificar y eliminar las aulas de esta institución.

Añadir una aula - desde esta sección puede añadir una nueva aula.
Solo son necesarios dos parámetros para añadir una nueva aula:

  • nombre - el nombre se ultiliza para identificar cada aula.
  • capacidad - la máxima capacidad de las aulas según el espacio disponible.

Editar aula - desde esta sección puede modificar o eliminar una aula que ya existe.
Esta es una lista de todas las aulas disponibles. Si no se han añadido aulas aparecerá un mensaje mostrando que no hay aulas disponibles.
Se pueden hacer dos acciones cambiando el valor de la combo de acción.

  • Actualizar aula - el nombre y la capacidad pueden ser modificados por cada fila de la lista pero solo puede ser actualitzado si esta acción es seleccionada en la combo de acción y el botón de acción es apretado.
  • Eliminar aula - La aula puede ser eliminada si esta acción es seleccionada y el botón de acción es apretado. Una pantalla le pedirá que confirme la acción.

Editar las direccions IP de la aula - desde esta sección será capaz de añadir el rango de direcciones IP usado en la aula.
En la primera sección usted podrá tomar las mismas acciones que desde la sección de Editar aula pero solo para la aula seleccionada. En la segona sección se muestra una lista de las direcciones IP añadidas (Si no se añadieron direcciones IP se mostrará un texto indicándolo ) y será capaz de añadir una IP al rango o quitárlo.

  • Añadir una IP - una dirección IP puede ser añadida desde la primera sección de esta segunda sección tecleándola y presionando el botón de añadir IP. Si la IP no tiene un formato correcto se indicará con un mensaje.
  • Quitar una IP - debajo la sección reservado para añadir es mostrado el rango IP. Se puede quitar cada una de ellas presionando el botón de Quitar IP.
