Editant les aules

Des d'aquesta pàgina pot afegir, modificar i eliminar les aules d'aquesta institució.

Afegir una aula - des d'aquesta secció pot afegir una nova aula.
Només són necessaris dos paràmetres per afegir una nova aula:

  • nom - el nom s'ultilitza per identificar cadascuna de les aules.
  • capacitat - la màxima capacitat de les aules segons l'espai disponible.

Editar aula - des d'aquesta secció pot modificar o eliminar una aula que ja existeix.
Aquesta és una llista de totes les aules disponibles. Si no s'han afegit aules apareixerà un missatge mostrant que no hi han aules disponibles.
Es poden fer dos accions canviant el valor de la combo d'acció.

  • Actualitzar aula - el nom i la capacitat poden ser modificats per cada fila de la llista però només pot ser actualitzat si aquesta acció és seleccionada a la combo d'acció i el botó d'acció és apretat.
  • Eliminar aula - l'aula pot ser eliminada si aquesta acció és seleccionada i el botó d'acció és apretat. Una pantalla li demanarà que confirmi l'acció.

Editar les adreces IP de l'aula - des d'aquesta secció se's capaç d'afegir un rang d'adreces IP utilitzades a l'aula.
A la primera secció es poden prendre les mateixes accions que a la secció d'edició de aules però nomès per l'aula sel·leccionada. A la segona secció és mostrada una llista de les adreces IP afegides (Si no s'han afegit adreces IP es mostrarà un text indicant-ho) i serà capaç d'afegir una IP al rang o esborrar-la.

  • Afegir una IP - una adreça IP pot ser afegida desde la primera secció d'aquesta segona secció, escrivint-la i prement el botó per Afegir IP. Si la IP no té un format correcte es mostrarà un missatge.
  • Esborrar una IP - a sota de la secció reservada per afegir una IP es mostrat el rang d'adreces IP. Es pot esborrar cadascúna d'aquestes prement el botó per esborrar IP.
