Modifier les salles de classe

Dès cette page on peut ajouter, modifier et éliminer les salles de classe de cette institution.

Ajouter une salle de classe - dès cette section on peut ajouter une nouvelle salle de classe.
Seulement sont nécessaires deux paramètres pour ajouter une nouvelle salle de classe:

  • prénom - le prénom s'utilise pour identifier chacune des salles de classe.
  • capacité - la maximum capacité de les salles de classe suivant l'espace disponible.

Modifier salle de classe - dès cette section on peut modifier ou éliminer une salle de classe qui existe déjà.
Cette est une liste de toutes les salles de classe disponibles. S'ils n'ont pas été ajoutées salles de classe apparaîtra un message en montrant qu'il n'y a pas des salles de classe disponibles.
On peut faire deux actions en changeant la valeur de la combo action.

  • Actualiser salle de classe - le prénom et la capacité peuvent être modifiés pour chaque rang de la liste mais seulement peuvent être actualisés si cette action est choisie à la combo d'action et le bouton d'action est poussé.
  • Éliminer salle de classe - la salle de classe peut être effacée si cette action est sélectionnée et le bouton d'action est poussé. Un écran lui demandera que confirme l'action.

Modifier des adresses IP pour la salle de classe - dès cette section on peut ajouter une rang d'adresses IP utilisés a la salle de classe.
Dans la premier section on peut prendre les mêmes actions que à la section d'Édition de salle de classe mais seulement pour le salle de classe selectionné. À la seconde section il y a une liste des adresses IP ajoutés (S'il n'y a pas des adresses IP disponibles un message sera montré en l'indiquant) et serrá capable d'ajouter une adresse IP au rang ou peut être effacée et possent le bouton pour ajouter adresse IP.   • Ajouter une adresse IP - une adresse IP peut être ajoutée au la premier section dans cette seconde section, ecrivent-la et possent le bouton pour ajouter adresse IP. S'il l'IP n'ai pas un format correct un message sera montré en l'indiquant.
  • Éliminer une adresseIP - en bas de la section reservée pour ajouter une IP le rang d'adresses est montrée. Chacque une peut être effacée si le bouton d'éffacer l'adresse c'est poussent.
