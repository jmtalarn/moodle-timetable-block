Modifier Groupes

Dès cette page on peut ajouter, modifier et éliminer les groupes et ses horaires pour chaque cours de cette institution.

Ajouter un groupe - dès cette section on peut ajouter un nouveau groupe en appuyant sur le bouton Ajouter un nouveau groupe .
Les paramètres nécessaires pour définir un groupe sont:

  • nom - le nom s'utilise pour identifier un groupe parmi les autres groupes.
  • nombre d'élèves - le nombre maximum d'élèves autorisés pour s'inscrire au groupe.
  • changement de groupe autorisé - le professeur peut décider si un élève peut s'effacer du groupe il même ou doit être le professeur qui l'efface du groupe.
  • groupe commun - un groupe commun est un groupe que une fois l'usager est immatriculé il peut s'inscrire au n' importe qu'autre.
    Pour un groupe non commun un usager seulement peut être enregistré à un de d'eux, et si l' usager choisit être enregistré à un autre groupe il será effacé du premier groupe.

Modifier groupe - dès cette section on peut modifier ou éliminer un groupe que déjà existe.
Cette est une liste de touts les groupes disponibles. S'il n'y a pas des groupes disponibles un message sera montré en l'indiquant.
Ils peuvent être déterminés quatre types d'actions en changeant la valeur de la combo d'action.

  • Actualiser groupe - Les paramètres nom, le nombre d'élèves, changement de groupe permis et groupe commun peuvent être modifiés pour chaque rang de la liste mais ils seulement peuvent être actualisés si cette participation est choisie à la combo participation et le bouton de participation est poussé.
  • Éliminer groupe - le groupe peut être éliminé si cette participation est selectionné et le bouton d' action poussé. Un message lui demandera que confirme l'action.
  • Ajouter tous les étudiants du cours - tous les étudiants du cours seront introduits à la liste une fois le bouton de participation est poussé. Un message lui demandera que confirme la participation.
  • Modifier une table d'heures du groupe - cette action te porte vers l' écran d' édition de l'horaire du groupe.

Modifier une table d'heures du groupe - dès cette section on peut modifier l'horaire du groupe choisit.
À la première section on peut faire les mêmes actions que à la section d'Édition de groupe mais seulement pour le groupe selectionné. À la seconde section il y a une représentation de la semaine ( jours de week-end en texte rouge) est montré avec toutes les heures que l'institution donne des cours. Pour chaque casier on peut faire une de cette deux actions:

  • Ajouter - Si cette action est possible un combo avec les salles de classe disponibles a l'institution pour cette heure sont montrées.
    Une classe peut être ajouter en choisissant la salle de classe approprié et en cliquant l'icône ajouter.
    Si la capacité de la salle de classe est plus petite que le nombre d'élèves que nous avions prévu, un message d'alerte en couleur bleu apparaîtra à la section supérieure et la classe sera ajoutée en texte bleu.
    Si la classe est ajoutée à un jour considéré week-end, un message d' alerte au couleur bleu apparaîtra à la section supérieure et la classe sera ajoutée en texte bleu.
  • Éliminer - Si une classe a été joindre on peut l'éliminer appuyent sur l'icône Éliminer.
