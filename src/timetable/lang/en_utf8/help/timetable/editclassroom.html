Editing Classroom

From this page you can add, modify and delete the classrooms in the institution.

Add a classroom - from this section you can add a new classroom.
Only two parameters are necessary for add a new classroom:

  • name - the name used for identify the classroom from each other.
  • capacity - the maximum classroom capacity considered for the space available.

Edit classroom - from this section you can modify or delete an existing classroom.
This is a list of all available classrooms. No classrooms available message wil be displayed if no classrooms were added.
Two actions can be token changing Action Commbo value

  • Update classroom - the name and capacity can be modified by each row in list but only will be updated if this action is selected in Combo Action and Action Button is pressed.
  • Delete classroom - the classroom will be deleted if this action is selected and Action Button is pressed. A confirm screen will ask you to confirm the action.

Edit classroom IP addresses - from this section you will be able to add the IP range used in the classroom.
In the first section you can take the same actions than in Edit classroom section but only for the classroom selected. In the second section a list of the IP addressess added will be shown (If no addresses were added a no IP addressess added text will be shown) and you will be able to add a IP to range or delete it.

  • Add an IP - an IP address can be added in the first section of this second section, typing it and pressing the Add IP button. If the IP hasn't a correct format a message will be shown.
  • Delete an IP - below the section reserved to add, the IP range is showed. You can remove each one by pressing the Delete IP button
