Editing Groups

From this page you can add, modify and delete the groups and his timetables for each course in the institution.

Add a group - from this section you can add a new group pressing the Add a new group button.
The parameters needed to well define a group are:

  • name - the name used for identify a group from each other.
  • students number - the maximum students number allowed to be joined to group.
  • group change allowed - the teacher can decide if a student can take off from the group himself or if only teachers can take off.
  • common group - a common group is a group that once enlisted the user can join any other.
    For a non common group a user can only be enlisted in one of them, and if joins another group it will be taked off from the first.

Edit group - from this section you can modify or delete an existing group.
This is a list of all available classrooms. No classrooms available message wil be displayed if no classrooms were added.
Two actions can be token changing Action Commbo value

  • Update group - the name,students number, change allowed and common group parameters can be modified by each row in list but only will be updated if this action is selected in Combo Action and Action Button is pressed.
  • Delete group - the group will be deleted if this action is selected and Action Button is pressed. A confirm screen will ask you to confirm the action.
  • Add all course students - all students in course will be enlisted once Action Button is pressed. A confirm screen will ask you to confirm the action.
  • Edit hourtable group - this action takes you to the group timetable editing screen.

Edit hourtable group - from this section you can modify the timetable for the selected group.
In the first section you can take the same actions than in Edit group section but only for the group selected. In the second section a week representation ( weekend days in red text) is showed with all hours that institution is giving classes. For each cell one of this two actions can be toke:

  • Add - If this action is possible a combo with the classrooms available in the institution for this hour is showed.
    A class can be added selecting the apropiate classroom and Add icon is pressed.
    If the classroom capacity is less than students number expected a warning message in blue is showed at the top section and class is added in blue text.
    If the class is added in a considered weekend day a warning message in blue is showed at the top section and class is added in blue text.
  • Delete - If a class was added you can delete pressing the Delete icon.
